# AppDelivery

Aplicativo que lista estabelecimentos de delivery de uma cidade, organizados por categoria, com contato direto via WhatsApp.

## Sobre

O app reúne os estabelecimentos de delivery de uma cidade em um só lugar. Cada estabelecimento é exibido em formato de card na tela inicial, organizado por categoria, e o usuário pode entrar em contato diretamente pelo WhatsApp.

## Funcionalidades

- Listagem de estabelecimentos em cards na tela inicial
- Organização por categoria
- Busca por nome ou categoria (sem precisar lembrar o nome completo)
- Contato direto com o estabelecimento via WhatsApp
- Cadastro de estabelecimento solicitado pelo dono, mediante registro
- Aba de suporte para dúvidas e problemas no uso

## Tecnologias

- Ionic
- TypeScript
- PHP (conexão com o banco de dados)

## Status

Projeto desenvolvido do zero como estudo. Funcional, porém em estágio inicial e sem manutenção contínua.

- Criação de um app do zero com Ionic
- Integração de front-end com banco de dados via PHP
